# 💻 Mac OS Portfolio Clone

A modern interactive portfolio inspired by macOS UI, built using React and modern frontend tools.

🔗 **Live Demo:** https://mac-os-ochre.vercel.app/

---

## 🚀 Features

* 🖥️ macOS-style draggable windows
* 📌 Dock system (open/close apps)
* 🪟 Multiple app windows:

  * GitHub Projects
  * Notes Viewer
  * Resume Viewer
  * Spotify Player
  * Calendar
  * Terminal (CLI-based portfolio)
* 🎯 Active window focus system
* 📱 Responsive design
* ⚡ Smooth UI interactions

---

## 🛠️ Tech Stack

* **Frontend:** React.js
* **Styling:** Tailwind CSS
* **Libraries:**

  * react-rnd (drag & resize windows)
  * react-terminal (CLI interface)
  * react-syntax-highlighter
* **Build Tool:** Vite
* **Deployment:** Vercel

## ⚙️ Installation & Setup

Clone the repository:

```bash
git clone https://github.com/tanveerahmed9413/mac-os.git
cd mac-os
```

Install dependencies:

```bash
npm install
```

Run locally:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
